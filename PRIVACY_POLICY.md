# Privacy Policy

Last updated: 2026

EFileOps Rename is a local desktop application designed to rename files on the user's computer.

## Information Collection

EFileOps Rename does not collect, store, transmit, or share any personal information.

All file processing is performed locally on the user's device.

## Data Usage

The application does not send any data to external servers and does not include analytics, telemetry, or tracking mechanisms.

## Third-Party Services

EFileOps Rename does not integrate with any third-party services.

## Changes

If this privacy policy changes in the future, updates will be published on this page.
